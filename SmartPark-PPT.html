üíª Implementation

Cost Calculation Algorithm:

function calculateCost(entry, exit, hourlyRate) {
  const [eh, em] = exit.split(':').map(Number);
  const [sh, sm] = entry.split(':').map(Number);
  const hours = (eh*60 + em - sh*60 - sm) / 60;
  return Math.ceil(hours) * hourlyRate;
}

Provides accurate cost calculations for user booking decisions
